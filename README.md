# 🎓 EduNexus
### Building the Future of Learning Systems

A full-stack course management platform built with:
- **Backend:** FastAPI + PostgreSQL + SQLAlchemy
- **Frontend:** React + Tailwind CSS + Vite
- **Auth:** JWT (JSON Web Tokens)
- **Docs:** Swagger UI at `/docs`

## Quick Start

1. Read **SETUP_GUIDE.md** for full step-by-step instructions
2. Setup PostgreSQL database named `edunexus`
3. Configure `backend/.env` with your database password
4. Run backend: `uvicorn app.main:app --reload`
5. Run frontend: `npm run dev`
